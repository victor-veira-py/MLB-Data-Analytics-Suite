# 📊 Phase 1: Individual Executive Report | Fase 1: Reporte Ejecutivo Individual

[English](#english-f1) | [Español](#español-f1)

---

## English
The first stage of the **MLB Analytics Suite** focuses on precision extraction. This module automates the retrieval of historical statistics for a single player directly from the official MLB API, transforming raw data into a professional corporate asset.

### ✨ Key Features:
* **Real-Time API Consumption:** Direct connection to official MLB endpoints for up-to-date data.
* **Career Stats Engine:** Automatic generation of summary rows with precise sums and career averages.
* **Executive Formatting:** Corporate blue styling, defined borders, and MLB-standard averages (e.g., .300 style).
* **Tech Stack:** Python, Pandas, Requests, and XlsxWriter.

---

## Español
La primera etapa de la **MLB Analytics Suite** se enfoca en la precisión de la extracción. Este módulo automatiza la obtención de estadísticas históricas de un jugador individual directamente desde la API oficial de la MLB, transformando datos crudos en un activo corporativo profesional.

### ✨ Características Principales:
* **Consumo de API en Tiempo Real:** Conexión directa a los endpoints oficiales de MLB para datos actualizados.
* **Motor de Estadísticas de Carrera:** Generación automática de filas de totales con sumatorias y promedios exactos.
* **Formato Profesional:** Estilo azul empresarial, bordes definidos y promedios estándar de MLB (ej. estilo .300).
* **Tecnologías:** Python, Pandas, Requests y XlsxWriter.

# 🤖 Phase 2: Bulk Report Automation | Fase 2: Automatización de Reportes Masivos

[English](#english-f2) | [Español](#español-f2)

---

## English
The second stage of the **MLB Analytics Suite** elevates the system to industrial-scale automation. This module enables the mass generation of individual statistical reports for a list of players, processing entire rosters or groups of prospects in seconds.

### ✨ Automation Capabilities:
* **Batch Processing (Bulk):** Iterates over a list of official MLB IDs and generates an independent Excel file for each player.
* **Format Consistency:** Every report maintains the executive standard (Corporate Blue, clean borders, and Career totals).
* **Dynamic Management:** Automatically names files based on the player's real name retrieved from the API.
* **Tech Stack:** Python, Pandas, XlsxWriter, and Requests (REST API).

---

## Español
La segunda etapa de la **MLB Analytics Suite** eleva el sistema a un nivel de automatización industrial. Este módulo permite la generación masiva de reportes estadísticos individuales para una lista de jugadores, procesando rosters completos o grupos de prospectos en segundos.

### ✨ Capacidades de Automatización:
* **Procesamiento por Lotes (Bulk):** Itera sobre una lista de IDs oficiales de MLB y genera un archivo Excel independiente para cada uno.
* **Consistencia de Formato:** Cada reporte mantiene el estándar ejecutivo (Azul corporativo, bordes limpios y fila de CARRERA).
* **Gestión Dinámica:** Crea nombres de archivos basados en el nombre real del jugador obtenido de la API.
* **Tecnologías:** Python, Pandas, XlsxWriter y Requests (API REST).

# 📊 Phase 3: Elite Consolidated Reports | Fase 3: Reportes Consolidados Élite

[English](#english-f3) | [Español](#español-f3)

---

## English
This is the most advanced stage of the **MLB Analytics Suite**. This engine processes a curated list of elite players and consolidates all their information into a **single professional Excel workbook** organized by dynamic tabs (sheets), providing a centralized executive dashboard.

### 🚀 Engineering Features:
* **Multi-Sheet Consolidation:** Generates a single Excel workbook (`.xlsx`) where each player has their own identified tab.
* **Special Character Handling:** Advanced logic to clean names (e.g., Acuña Jr) to prevent Excel naming errors.
* **Aggregate Calculations:** API duplicate filtering and generation of **CAREER** totals with precise averages.
* **High-Visibility Design:** Navy blue headers (#1F4E78), highlighted totals, and defined borders.
* **Tech Stack:** Python, Pandas (Complex Data), XlsxWriter, and Requests.

---

## Español
Esta es la etapa más avanzada de la **MLB Analytics Suite**. Este motor procesa una lista curada de jugadores élite y consolida toda la información en un **único archivo Excel profesional** organizado por pestañas dinámicas, entregando un tablero ejecutivo centralizado.

### 🚀 Características de Ingeniería:
* **Consolidación Multi-hoja:** Genera un solo libro de Excel (`.xlsx`) donde cada jugador tiene su propia pestaña identificada.
* **Manejo de Caracteres Especiales:** Lógica avanzada para limpiar nombres (ej. Acuña Jr) y evitar errores en las etiquetas de las hojas.
* **Cálculos Agregados:** Filtrado de duplicados de la API y generación de filas de **CARRERA** con promedios exactos.
* **Diseño de Alta Visibilidad:** Cabeceras azul marino (#1F4E78), totales resaltados y bordes definidos.
* **Tecnologías:** Python, Pandas (Datos complejos), XlsxWriter y Requests.
